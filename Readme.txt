Saya membuat platform dashboard untuk menganalisis data penyewaan sepeda. Proyek ini dibuat untuk memenuhi tugas GDGoC Assigment Machine Learning.

A. Struktur File:
dashboard.py: File yang terhubung ke streamlit
day.csv:Data set penyewaan sepeda yang dianalisis
requirement.txt: Daftar library

B. Hasil Analisis (Insight)
-Trean Harian: Terjadi jumlah kenaikan yang signifikan dari tahun 2011 ke 2012
-Kondisi Hari: Menunjukan pada hari kerja lebih banyak jumlah sewa sepeda dibandingkan hari libur.Hal tersebut menunjukan sewa sepeda banyak dipakai pada rutinitas harian (hari kerja/sekolah/kuliah)

C. Menjalankan secara lokal 
-Pastikan python sudah terinstall
-Instal library:pip install pandas matplotlib seaborn streamlit

D. Jalankan Aplikasi di streamlit:
Di Terminal, ketik: pip install pandas matplotlib seaborn streamlit
